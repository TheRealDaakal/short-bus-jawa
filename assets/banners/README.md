# Raid Banners

Drop banner images here to have them automatically show up on raid boards
for that content. No code changes needed - just add the file.

## Naming

Filename must match the content's name, lowercased, with spaces and
punctuation replaced by underscores. Supported formats: .png, .jpg,
.jpeg, .webp

## Expected filenames (current content)

### Operations
- eternity_vault
- karaggas_palace
- explosive_conflict
- terror_from_beyond
- scum_and_villainy
- dread_fortress
- dread_palace
- temple_of_sacrifice
- gods_from_the_machine
- the_nature_of_progress
- r_4_anomaly

### Lair Bosses
- xenoanalyst_ii
- the_eyeless
- golden_fury
- colossal_monolith
- geonosian_hive_queen
- relentless_replication_propagator_core_xr_53

Recommended size: 1200x400px or similar wide banner aspect ratio (Discord
embeds display images up to ~1600px wide). Any size works, but very tall
images will make the embed awkwardly long.
